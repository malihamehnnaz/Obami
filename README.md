# OBA'MI Website

This is the modern, clean, and professional website for OBA'MI, a Vietnamese restaurant in Surry Hills, Sydney.
Built with [Next.js](https://nextjs.org), [Tailwind CSS](https://tailwindcss.com), and [Lucide React](https://lucide.dev).

## Features

-   **Modern Design**: Clean and professional look with a focus on user experience.
-   **Responsive**: Works great on desktop, tablet, and mobile.
-   **Menu Showcase**: Categorized menu with images and descriptions.
-   **Online Ordering**: Direct links to Uber Eats and DoorDash.
-   **Gallery**: Visual showcase of the restaurant and dishes.
-   **Contact Info**: Easy access to location, hours, and contact details.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

-   `src/app`: App Router pages (Home, Menu, About, Contact, Gallery).
-   `src/components`: Reusable components (Navbar, Footer).
-   `public`: Static assets.

## Customization

-   **Images**: Replace placeholder images in the code with real assets.
-   **Content**: Update text content in the respective page files.
-   **Colors**: Customize the color scheme in `tailwind.config.ts` or `globals.css`.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
